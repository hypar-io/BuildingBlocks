# Site By Sketch

Creates a Site from a polygon sketch.

|Input Name|Type|Description|
|---|---|---|
|Perimeter|https://schemas.hypar.io/Polygon.json|Perimeter of the building envelope.|


<br>

|Output Name|Type|Description|
|---|---|---|
|Area|Number|Total site area.|
